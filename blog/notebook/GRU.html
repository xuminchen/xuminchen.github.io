# GRU

# Introduction

Gate Recurrent Unit，同样是RNN的变种，与LSTM类似，都是通过门控单元来实现对信息的保留。不过GRU的构造比LSTM更加简单，只有两个门，综合起来计算量比LSTM小很多。

在工业界上，更倾向于GRU，因为效果与LSTM相近，但是计算开销小。

# 门控单元

这里只有两个门，分别是Update Gate和Reset Gate

# Update Gate更新门

表示在t时刻要抽取多少信息保存到hth_th​t​​中

u(t)=σ(Wuht−1+Uuxt+bu)u^{(t)}=\sigma(W_uh_{t-1}+U_ux_t+b_u) u​(t)​​=σ(W​u​​h​t−1​​+U​u​​x​t​​+b​u​​)

# Reset Gate重置门

与LSTM中forget gate的作用一致，表示保留多少或放弃多少信息

r(t)=σ(Wrht−1+Urxt+br)r^{(t)}=\sigma(W_rh_{t-1}+U_rx_t+b_r) r​(t)​​=σ(W​r​​h​t−1​​+U​r​​x​t​​+b​r​​)

# Hidden State

在t时刻额外得到的信息

h~(t)=tanh(Wh⋅(r(t)ht−1)+Uhxt+bh)\tilde{h}^{(t)}=tanh(W_h\cdot(r^{(t)}h_{t-1})+U_hx_t+b_h) ​h​~​​​(t)​​=tanh(W​h​​⋅(r​(t)​​h​t−1​​)+U​h​​x​t​​+b​h​​)

最终的模型输出：

ht=(1−u(t))h(t−1)+u(t)h~(t)h_t=(1-u^{(t)})h^{(t-1)}+u^{(t)}\tilde{h}^{(t)} h​t​​=(1−u​(t)​​)h​(t−1)​​+u​(t)​​​h​~​​​(t)​​

其中，u(t)u^{(t)}u​(t)​​在这里可以看作是一个加权平均的作用。
