# LSTM

# Introduction

Long Short-Term Memory, 在传统的RNN基础上优化修改，引入门控机制，选择性地存储信息，使得在有限的模型空间内保留更多的信息。

# 门控单元

与传统RNN不一样的是引入了一个新的Cell state，用来保存过去的信息。而门控单元则是用来控制信息的流向，控制信息是否保存到cell state之中。

LSTM一共有三个门：遗忘门、输入门、输出门。

# 遗忘门

forget gate，表示要不要记忆一些信息或者遗忘掉一些之前的信息。

f(t)=σ(Wfht−1+Ufxt+bf)f^{(t)}=\sigma(W_fh_{t-1}+U_fx_t+b_f) f​(t)​​=σ(W​f​​h​t−1​​+U​f​​x​t​​+b​f​​)

由于使用了sigmoid激活函数，使得输出的范围在0-1之间，可以理解为遗忘掉百分之多少的信息，其中，0表示全忘掉，1表示全保留。

# 输入门

input gate，表示保留或者提取输入的多少信息。

i(t)=σ(Wiht−1+Uixt+bi)i^{(t)}=\sigma(W_ih_{t-1}+U_ix_t+b_i) i​(t)​​=σ(W​i​​h​t−1​​+U​i​​x​t​​+b​i​​)

# 输出门

output gate, 表示有多少信息输出到下一个单元。

o(t)=σ(Woht−1+Uoxt+bo)o^{(t)}=\sigma(W_oh_{t-1}+U_ox_t+b_o) o​(t)​​=σ(W​o​​h​t−1​​+U​o​​x​t​​+b​o​​)

# cell state

cell state用来存储文本中的长时间依赖。在t时刻，当前单元会额外拿到一些信息：

C~(t)=tanh(Wcht−1+Ucxt+bc)\tilde{C}^{(t)} = tanh(W_ch_{t-1}+U_cx_t+b_c) ​C​~​​​(t)​​=tanh(W​c​​h​t−1​​+U​c​​x​t​​+b​c​​)

而在t时刻时，单元输出的真正的cell state：

C(t)=f(t)C(t−1)+i(t)C~(t)C^{(t)}=f^{(t)}C^{(t-1)}+i^{(t)}\tilde{C}^{(t)} C​(t)​​=f​(t)​​C​(t−1)​​+i​(t)​​​C​~​​​(t)​​

其中，前面一项表示从上一时刻中丢弃多少信息，而后面一项表示的是当前t时刻要保留多少信息。

# hidden layer

最后的hidden layer输出是：

ht=o(t)tanh(C(t))h_t=o^{(t)}tanh(C^{(t)}) h​t​​=o​(t)​​tanh(C​(t)​​)

这个hth_th​t​​即是传入到下一个单元的hidden state，也是当前单元的最终输出。

另外，Lstm在计算过程中采用的激活函数(tanh和sigmoid)并不是固定的，只是作者在提出这个框架时，做了一些实验，发现采用这两个激活函数的效果最好。

# 优缺点

Pros

  1. 在一定程度上，解决了梯度消失的问题。但是并没完全解决。
  2. 提供了长文本依赖的解决方案，可以捕获到较长文本的信息。

Cons

  1. 不能100%保证梯度的问题得到解决。
