# 文本预处理

# Introduction

文本，即是预料，带有文字描述性的文本都可以归类于预料。但是原始的文本或预料是无法直接使用到模型之中，需要进行预处理操作。

目前面试都会问到一个问题：对项目的文本进行了什么样的预处理操作。

文本预处理的方法主要包括数据清洗、分词、词性标注、去停用词等。

# 文本清洗

文本清洗(数据清洗)即是保留预料中有用的文本数据，删除噪音数据（但是不可能完全删除掉）。常见的清洗方式有：人工去重、对齐、删除、标注等。

而噪音数据，包括：标点符号、重复文本、错误值、缺失值、异常值等。

# 分词

数据清洗完后，才进入到分词的阶段。常用的分词方法有基于规则的，基于统计的，而统计的样本内容来自于一些标准的语料库。

常用的分词库有：jieba、HanLP、LTP。

如果没有对分词任务的特殊需求，可以直接采用现成的分词工具，否则要自己开发。

# 词性标注

指为分词结果中的每个token标注正确的词性，即确定每个词是名词、动词、形容词或是其他词性的过程。

词性标注的作用:

  1. 消消除歧义，除同一词语在不同语境或用法下的不同意思。
  2. 强化基于单词的特征。在京东项目一中采用该方法，通过统计句子中每个词的词性出现次数除以句子长度。
  3. 标准化
  4. 词形还原
  5. 移除停用词。

常用的词性标注方法是基于规则的、也有基于统计的。比如：最大熵词性标注、HMM词性标注等。

# 移除停用词

停用词即是在文本中，不论任何语言，在一个句子中，出现的作连接作用的词语，这些词语没有具体的意义。

常用的停用词表有：哈工大停用词表、百度停用词表等。

常用中文停用词表stopwords: https://github.com/goto456/stopwords

# 词频统计

即统计分词后文本的词频，目的是找出对文本影响最大的词汇，是文本挖掘的重要手段。统计这些词出现的频率可以帮助我们了解文章重点强调了什么，进而方便后续构建模型。
